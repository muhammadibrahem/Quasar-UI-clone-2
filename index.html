General

What is the all-access pass?

It gives you access to all the QuasarUI components and features.

Is there a recurring fee?

No, it is a one-time payment with lifetime access.

How do I access the QuasarUI components?

You can access them after purchasing the pass from your dashboard.

What is Quasar?

Quasar is a Vue.js framework that allows you to build high-performance apps.

What is the difference between QuasarUI and Quasar?

Quasar is a framework; QuasarUI is a design system built with Quasar.

What does the lock symbol mean?

It means the feature or component is restricted to premium users.

Licensing

Do I have to purchase a license for each project I use these QuasarUI components in?

No, you can use QuasarUI components in as many projects as you like and unlimited amount of times.

Can I use these QuasarUI components in Commercial projects?

Yes, you can use these componets in personal and commercial projects.

Can I modify the QuasarUI components?

Yes, you can modify the QuasarUI componets to fit your project needs.

Can I use these QuasarUI components in an Open Source project?

Yes! As long as you are building a project that is open source and not a direct competitor to QuasarUI componets.

Can I sell templates or themes that are built using these QuasarUI components?

No, you cannot sell templates or themes that are built using these QuasarUI componets. You can use them in your projects, but you cannot resell them as standalone products.

Compatibility

What versions of Quasar are these components compatible with?

QuasarUI components are developed using the latest version of Quasar, which is currently v2.16.6.

What browsers are supported?

All evergreen browsers are supported, including Chrome, Firefox, Safari, and Edge. For more detailed information, please refer to the official documentation.

Do any of the packages come with Figma, Sketch, or Adobe XD files?

No, the packages only include the source code for the components.
